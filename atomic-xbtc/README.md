# Swaps DAO

## Clarigen

Project adds [Clarigen](https://www.clarigen.dev/docs/getting-started) to Friedger swap contracts. 

```bash
npm init
npm install --save-dev @clarigen/cli
npm install -g pnpm
npm i @clarigen/test
npm i @clarigen/core
clarigen init
npx clarigen
npx clarigen --watch
```

## Tests

```bash
npm run test
npm run watch
```
